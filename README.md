# Random Users

1. Get a set of Random Users.
2. Concatenate the full name of the users based on first and last names.
3. Calculate the average age of the users.

## Setup

```bash
# install deps
yarn
# build the code
yarn build
```

## Usage

```bash
yarn run-build
# or
yarn build
node build/index.js
```
